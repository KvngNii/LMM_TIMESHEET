# Lavitta Marketing Management — Time Tracking

Employee time tracking platform for Lavitta Marketing Management. Built with Next.js 15, TypeScript, and Tailwind CSS.

## Features

- **Dashboard** — Real-time overview with charts, stats, and team activity feed
- **Clock In/Out** — Live timer with break tracking and project selection
- **Timesheets** — Filterable, paginated timesheet management with approve/reject workflow
- **Employees** — Employee directory with stats, search, and CRUD operations
- **Reports** — Rich analytics with hours trends, project breakdowns, department costs, and employee rankings
- **Settings** — Profile, notifications, preferences, and company configuration

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
